# Trust

## Purpose

Trust is the feature layer that makes the product believable. It combines confidence indicators, explanation, lineage, and clear warning behavior.

## Core Capabilities

- expose forecast confidence
- explain major inputs and changes
- distinguish stable state from uncertain state
- preserve user trust when data is stale or incomplete
