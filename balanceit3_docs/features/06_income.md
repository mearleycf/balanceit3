# Income

## Purpose

Income scheduling models expected inflows so the forecast can anticipate recovery points, protect against overly conservative spending, and account for timing drift.

## Core Capabilities

- create expected income schedules
- support common cadence patterns
- track next expected date and amount
- compare expected income against actual deposits

## Key Requirements

- `FR-12`
- `FR-14`
- `FR-15`
- `FR-16` through `FR-19`

## Risks

- late or missed income can make the safe-to-spend amount dangerously optimistic if not handled conservatively
