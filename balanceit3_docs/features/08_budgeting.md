# Budgeting

## Purpose

Budgeting helps the household define discretionary limits and contextualize how much of the forecast buffer should be treated as intentionally spendable.

## Core Capabilities

- define category or group targets
- compare actual discretionary spend versus plan
- expose budget pressure without hiding raw forecast state

## Key Requirements

- `FR-23` through `FR-25`

## Scope Note

Budgeting supports safe spending guidance, but it is subordinate to the core forecasting and safe-to-spend calculation.
