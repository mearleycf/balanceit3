# User Story: Understand Forecast Warnings

## Story

As a household user,
I want warnings to explain when the forecast is risky or uncertain,
So that I know whether the current `safe to spend` amount is reliable.

## Source Inputs

- feature dossier: `features/09_alerts.md`
- FR references: `FR-21`, `FR-30`, `FR-31`

## Acceptance Notes

- warning severity is clear
- copy is understandable for non-technical use
- admins can route from an alert to the corrective workflow
